Bienvenido a mi sitio web

Sistema de reporte de clases de la universidad

Mi proyecto

Mi proyecto consiste en un sistema web que permite a los profesores y estudiantes de la universidad reportar y consultar las clases que se han realizado, asi como tambien la posibilidad de gestionar el personal universitario. El sistema web es fácil de usar, seguro y eficiente. Con este sistema, se puede mejorar la calidad de la educación y la comunicación entre los miembros de la comunidad universitaria.

Demostracion

A continuacion le mostraremos una pequeña demostracion de algunos codigos utilizados y el funcionamiento basico del proyecto. Si te interesa probarlo al completo puedes contactarnos por alguna de nuestras redes y te enviaremos el proyecto junto a la base de datos para que lo pruebes y utilices.

Mi equipo

Mi equipo está conformado por: (Marlon Bolaño),(Sebastian Monsalve),(Carlos Viloria) con el desarrollo de la base de datos. Cada uno tiene un rol y una función específica dentro del equipo, y juntos trabajamos para ofrecer un producto de calidad y satisfacer las necesidades de nuestros clientes.

Marlon Bolaño

Se encargo de crear la base de datos en mySQL, para poder guardar los datos que necesita ingresar el usuario, como: el inicio de sesion, registrar usuarios, registrar las clases...

Carlos Viloria

Se encargo de crear las interfacez graficas en JAVA y asi tener una estructura de donde poder ingresar los datos, poder modificarlos y También poder eliminar algunos datos ingresado.

Sebastian Monsalve

Se encargo de conectar la intefaz grafica y la base de datos de mySQL y asi una vez ingresados los datos en la interfacez quedan almacenas en la base de datos.
